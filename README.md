# Udemy-Homepage-Clone
This project is a front-end clone of the Udemy homepage, created using pure HTML and CSS.
It mainly focuses on understanding and applying Flexbox layout techniques to build a clean, organized, and responsive structure.

**Project Overview**

Built using basic HTML and CSS

Uses Flexbox for layout and alignment

Includes properties like:

   display: flex

   flex-wrap

   justify-content

   align-items

 Single-page design

User-friendly structure and simple visual design

**Purpose of the Project**

The goal of this project is to practice:

Page structuring using Flexbox

Aligning elements efficiently

Creating a clean homepage layout

Improving front-end skills with basic technologies

**Features**

Simple and clean UI

Responsive layout using Flexbox

Organized sections similar to Udemy’s homepage

Easy-to-understand code structure

Beginner-friendly design

**How to View the Project**

Download the repository

Open the file: index.html

The webpage will open in your browser
