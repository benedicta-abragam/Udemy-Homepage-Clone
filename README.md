# Udemy‑Homepage‑Clone
 A static HTML & CSS clone of the Udemy homepage.

## Overview  
This project recreates the homepage layout of Udemy using only *HTML5 and CSS3* — focusing on responsive design, layout structure, and visual styling.

## Features  
- Static homepage layout: header, navigation bar, banner section  
- Course listings / cards  mimicking Udemy style  
- Responsive design using CSS (Flexbox / Media Queries)  
- No JavaScript or backend — pure HTML & CSS  

## Usage  
1. Clone the repository:  
   git clone https://github.com/benedicta-abragam/Udemy-Homepage-Clone.git

2. Open index.html in your web browser to view the homepage clone.

## Built With
HTML5
CSS3 (Flex/Media Queries)
